# NodeRed-PVBatt-control
Node-RED PV and Battery Voltage Relay Control flow

Allows user to set thresholds for battery discharge and cut off levels as well as incoming PV voltage level levels to control the state of a Victron relay.\nThe intended purpose is to limit the on state of the relay, and therefore the running time of a large appliance on a sailboat to be limited to cases where the batteries are charged and the sun is shining.
